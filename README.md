# 장병들을 위한 스케줄러, 커뮤니티 서비스 Hymn
## Description
외출을 나가고, 휴대전화를 써도 장병들에게 가장 관심이 가고 오매불망 바라는 건 전역일 겁니다. (100퍼)
이제 핸드폰에 당당하게 자기 (전역 및) 휴가 스케줄 관리하고, 서로 잡담도 자유롭게 나눌수 있는 앱이 있으면 좋겠다고 생각했습니다.
## Features
### Scheduler
- Daily
  - 실근무 계산기
  - TODO List + Reminder
- Weekly
  - 실근무 계산기
  - TODO List + Reminder
- Monthly
- Military-service
### Community
- 관심사 기반 커뮤니티
  - 자유롭게 관심사로 게시판을 제작할 수 있고, 관리자가 될 수 있습니다.
